# 🎓 PreSkool - Système de Gestion Scolaire (Django)

Ce projet est une plateforme de gestion scolaire développée dans le cadre du **Projet de Fin de Module (PFM)** à la **Faculté des Sciences et Techniques (FST) de Tanger**.

## 🛠️ Technologies Utilisées
* **Backend :** Python & Django Framework
* **Base de données :** SQLite (Par défaut)
* **Frontend :** HTML5, CSS3, Django Templates
* **Gestion de version :** Git & GitHub
